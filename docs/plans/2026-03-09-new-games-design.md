# 4개 신규 게임 추가 디자인

**날짜**: 2026-03-09
**접근법**: A. 병렬 일괄 구현

## 공통 사항

- 순수 HTML/CSS/JS, 단일 파일
- `balance-game.html` 디자인 시스템 계승 (CSS 변수, 다크모드, 토퍼바)
- `shared/transitions.css` + `shared/transitions.js` 포함
- 허브 복귀 링크: `← 게임 목록` → `index.html`
- Fisher-Yates 셔플
- 모바일 우선 반응형

## 1. 진실 or 도전 (`truth-dare.html`)

- **흐름**: 난이도 선택(순한맛/매운맛/불닭맛) → 카드 뷰
- **카드**: 탭하면 진실(파란)/도전(빨간) 랜덤 결정 → 미션 표시
- **네비게이션**: 이전/다음/셔플
- **콘텐츠**: 카테고리 5개 × 난이도 3단계 × 각 5개 = 75개+
- **색상**: `--card-color: #ef4444` (red)

## 2. 랜덤 벌칙 룰렛 (`roulette.html`)

- **흐름**: 참가자 설정(이름 입력 or 번호) → 룰렛 스핀
- **UI**: 이중 슬롯머신 (누가 + 뭘)
- **버튼**: "돌리기!" → 슬롯 회전 애니메이션 → 결과
- **강도**: 순한맛/매운맛 토글 (불닭맛 제외)
- **콘텐츠**: 60개+ 벌칙 (순한맛 30, 매운맛 30)
- **색상**: `--card-color: #f59e0b` (amber)

## 3. 스피드 퀴즈 (`speed-quiz.html`)

- **흐름**: 카테고리 선택 → 타이머(30/60/90초) → 플레이
- **UI**: 큰 단어 + 카운트다운 + 금칙어 2-3개
- **버튼**: 정답!/패스
- **결과**: 점수 + 정답/패스 단어 목록
- **콘텐츠**: 6카테고리 × 20단어 = 120개+
- **색상**: `--card-color: #8b5cf6` (violet)

## 4. 라이어 게임 (`liar-game.html`)

- **흐름**: 참가자 수 → 카테고리 → 역할 배정(폰 돌려보기) → 토론 타이머 → 투표 → 결과
- **역할 배정**: 탭하면 역할 표시/숨김 (시민: 주제, 라이어: "당신은 라이어")
- **투표**: 참가자 버튼 클릭으로 투표
- **결과**: 라이어 공개 + 투표 결과 애니메이션
- **콘텐츠**: 6카테고리 × 15주제쌍 = 90개+
- **색상**: `--card-color: #06b6d4` (cyan)

## 허브 업데이트

- `index.html` 게임 수: 4 → 8
- 4개 신규 게임 카드 추가
